# Dynamic Economy

Dynamic Economy adds a supply-and-demand based market system to Stardew Valley.

## Features

- **Dynamic Pricing**  
  Item prices change based on supply and demand.

- **NPC Demand Simulation**  
  Villagers influence demand based on their preferences.

- **Seasonal Modifiers**  
  Different seasons affect item demand and pricing.

- **Market UI**  
  Press `F6` to toggle a simple in-game overlay showing market data.

- **Multiplayer Support**  
  Economy is synced across players (host-controlled).

## How It Works

- Selling items increases supply → prices go down  
- NPC demand increases value → prices go up  
- Market gradually stabilizes over time  

## Requirements

- SMAPI (Stardew Modding API)

## Installation

1. Install SMAPI
2. Copy the mod folder into `Stardew Valley/Mods`
3. Run the game using SMAPI

## Notes

- Compatible with most mods
- Mods that change item prices may conflict
